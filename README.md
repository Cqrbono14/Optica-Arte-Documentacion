# Documentación – Sistema de Inventario para Óptica Arte

## Descripción

Este repositorio contiene la **documentación y evidencias académicas** del proyecto *Sistema de Inventario para Óptica Arte*, desarrollado como parte de los requisitos para la materia de Administración de Software y Proyectos, cursada durante 8vo Semestre (Enero 2026 al Junio 2026) de la carrera de ingeniería en computación inteligente, disponible en la Universidad Autónoma de Aguascalientes.

El objetivo de este repositorio es **organizar las actividades realizadas durante el curso y las rúbricas de evaluación correspondientes**, con el fin de cumplir con los entregables solicitados.

El **código fuente del sistema se encuentra en un repositorio separado y privado**, por lo que no se incluye en este repositorio.

---

## Estructura del repositorio

```
Optica-Arte-Documentacion/
│
├── Actividades/
│   └── Evidencias y entregables de cada actividad realizada durante el curso
│
├── Rubricas/
│   └── Rúbricas de evaluación proporcionadas para cada actividad
│
└── README.md
```

---

## Contenido

### Actividades

En esta carpeta se almacenan los documentos, reportes y materiales generados para cada una de las actividades relacionadas con el desarrollo del proyecto.

### Rúbricas

Contiene las rúbricas de evaluación utilizadas para calificar cada actividad del curso.

---

## Propósito del repositorio

* Organizar las evidencias académicas del proyecto.
* Mantener un registro claro de las actividades realizadas.
* Facilitar la revisión del trabajo desarrollado durante la materia.

---

## Proyecto

El proyecto documentado corresponde a un **sistema de control de inventario para una óptica**, el cual permite registrar armazones disponibles, controlar el stock y almacenar notas asociadas a clientes.
